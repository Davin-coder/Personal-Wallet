## Descripción

**Personal Wallet** es una aplicación de consola desarrollada en Python que permite llevar un registro y control de tus finanzas personales. Puedes consultar tu saldo, revisar el historial de movimientos y realizar transacciones de forma rápida sin necesidad de interfaces gráficas.

---

## Características

| # | Funcionalidad | Descripción |
|---|---------------|-------------|
| 1 | Ver saldo neto | Consulta tu balance actual disponible |
| 2 | Ver historial | Revisa todos los movimientos realizados |
| 3 | Realizar una transacción | Registra ingresos o gastos |
| 4 | Retirar dinero | Registra un retiro de fondos |

## Instalación y Ejecución

### Requisitos previos

- Tener instalado [Python 3.10+](https://www.python.org/downloads/)

### Clonar el repositorio

```bash
git clone 
cd personal_wallet
```

### Ejecutar la aplicación

```bash
python src/main.py
```
